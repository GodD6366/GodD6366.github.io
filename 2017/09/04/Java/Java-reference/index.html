Java的四种引用类型

发表于 2017-09-04 | | 阅读次数:

Java 提供四种引用类型:强引用,软引用,弱引用,虚引用

强引用

如下直接 new 对象出来的引用方式是强引用,强引用会出现 OutOfMemoryError 异常,因为GC 永远不会回收此类对象的内存

软引用

如果内存足够,GC 是不会回收此类型引用的,但是当内存不足时,GC 会开始回收这些对象的内存.
所以配合 ReferenceQueue,可以实现当我们在用软引用的时候，可以去查询这个队列，来知道哪些对象已经被删除了，如果发现这个对象被删除了，我们可能会重新创建一个对象,或者执行对应的业务。
